Getting Started

VS Code is a powerful yet lightweight code editor used for many programming languages.

What You Can Do Here:

  • Install the editor
  • Explore the interface
  • Open your first project

Quick Guide:

  1. Download & Install
    Get VS Code from the official website and run the installer.
  2. Open a Folder
    Go to File → Open Folder to start working on your files.
  3. Basic Interface
    You will see the Activity Bar, Side Bar, Editor Area, and Status Bar.

Tip: Use the Command Palette (Ctrl + Shift + P) to access most features quickly.

Editor Basics

The editor is where you write and modify your code. VS Code provides helpful tools even without extensions.

Key Features:

  • IntelliSense: Auto-complete suggestions
  • Syntax Highlighting: Colors for keywords and variables
  • Tabs: Open multiple files
  • Split View: Work side-by-side

Useful Tools:

  • Search (Ctrl + F): Quickly find text
  • Replace (Ctrl + H): Replace text across the file
  • Multi-cursor:
    • Alt + Click
    • Ctrl + Alt + Up/Down

Tip: Use Ctrl + / to quickly comment or uncomment lines.

Extensions

Extensions allow you to customize your coding environment.

Why Use Extensions?

  • Add support for languages like Python, Java, C++, PHP
  • Improve formatting
  • Add debugging tools
  • Change themes and icons

How to Install:

  1. Click the Extensions icon on the sidebar.
  2. Type the extension name.
  3. Click Install.

Popular Extensions:

  • Prettier → automatic code formatting
  • Live Server → auto-refresh browser
  • ESLint → code quality checker
  • Python → debugging + IntelliSense
Keybindings and Shortcuts

Shortcuts make coding faster and smoother.

Most Useful Shortcuts:

  • Ctrl + P → quick file search
  • Ctrl + ` → toggle terminal
  • Ctrl + B → show/hide sidebar
  • Ctrl + Shift + P → command palette
  • Alt + Up/Down → move line

Editing Shortcuts:

  • Ctrl + C / Ctrl + X: copy or cut lines without selecting
  • Shift + Alt + Down: duplicate line

Customizing: Go to File → Preferences → Keyboard Shortcuts to modify any keybinding.

Debugging and Run

VS Code includes built-in debugging tools for many languages.

What You Can Do:

  • Start and stop debugging
  • Set breakpoints
  • Inspect variables
  • View call stack
  • Read console output

What You Can Do:

  1. Click Run and Debug from the sidebar.
  2. Select a debugging environment (e.g., Python, Node.js).
  3. Press F5 to start.

Breakpoints: Click next to a line number to enable or disable a breakpoint.

Tip: Use Debug Console to run commands and view program output.
